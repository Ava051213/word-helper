# 单词记忆助手使用手册

## 简介

单词记忆助手是一个基于艾宾浩斯遗忘曲线理论的智能单词记忆系统。它通过科学的时间间隔安排单词复习，帮助用户更高效地记忆单词。

## 安装指南

### 系统要求
- Python 3.6 或更高版本
- Windows/Linux/macOS 操作系统

### 安装步骤
1. 确保已安装 Python 3.8+。
2. 克隆或下载项目代码。
3. 运行 `pip install -r requirements.txt` 安装依赖。
4. 运行 `start_gui.bat` (图形界面) 或 `start_cli.bat` (命令行界面)。

## 界面版本说明

本系统提供两个版本的用户界面：

### 命令行版本 (CLI)
- 轻量级，资源占用少
- 适合喜欢键盘操作的用户
- 启动快速，响应迅速

### 图形界面版本 (GUI)
- 直观易用，可视化操作
- 提供丰富的功能和更好的用户体验
- 支持多种视图和交互方式

## 命令行版本功能详解

### 1. 添加单词
- 选择主菜单中的"1. 添加单词"
- 输入单词、释义、例句（可选）和分类（可选）
- 系统会自动保存单词信息

### 2. 查看单词
- 选择主菜单中的"2. 查看单词"
- 系统会显示所有已添加的单词列表

### 3. 复习单词
- 选择主菜单中的"3. 复习单词"
- 系统会根据艾宾浩斯记忆曲线安排需要复习的单词
- 对于每个单词，确认是否认识
- 系统会根据回答正确与否调整下次复习时间

### 4. 搜索单词
- 选择主菜单中的"4. 搜索单词"
- 输入关键词（单词、释义、例句或分类）
- 系统会显示所有匹配的单词

### 5. 删除单词
- 选择主菜单中的"5. 删除单词"
- 输入要删除的单词
- 确认删除操作

### 6. 学习统计
- 选择主菜单中的"6. 学习统计"
- 查看总单词数、已复习单词数等统计信息

## 艾宾浩斯记忆曲线原理

艾宾浩斯遗忘曲线是由德国心理学家赫尔曼·艾宾浩斯提出的关于遗忘规律的理论。该理论指出：

1. 遗忘的速度先快后慢
2. 及时复习可以有效减缓遗忘
3. 科学安排复习时间可以最大化记忆效果

本系统采用以下复习间隔：
- 第1次复习：1天后
- 第2次复习：2天后
- 第3次复习：4天后
- 第4次复习：7天后
- 第5次复习：15天后
- 第6次复习：30天后
- 之后每次间隔翻倍

## 数据存储与管理

### 数据存储
系统将单词数据保存在 `data/words.json` 文件中，采用JSON格式存储，便于查看和备份。

### 数据导入/导出
- 支持CSV格式的数据导入和导出
- 可与其他学习工具进行数据交换
- 提供批量操作功能

### 数据备份与恢复
- 自动定期备份数据
- 支持手动备份
- 提供一键恢复功能

start_gui.bat

# 启动改进版图形界面
start_gui_improved.bat
```

### macOS/Linux系统

./start_cli.sh

# 启动标准图形界面
./start_gui.sh

# 启动改进版图形界面
./start_gui_improved.sh
```

python src/main.py

# 启动标准图形界面
python src/gui_main.py

# 启动改进版图形界面
python src/gui_main_improved.py

## 常见问题

### 1. 如何备份我的单词数据？
- 方法一：直接复制 `data/words.json` 文件
- 方法二：使用图形界面的备份功能
- 方法三：导出为CSV文件

### 2. 如何恢复单词数据？
- 方法一：将备份的 `words.json` 文件复制到 `data/` 目录下覆盖原文件
- 方法二：使用图形界面的恢复功能
- 方法三：从CSV文件导入

### 3. 忘记某个单词的意思怎么办？
- 可以使用搜索单词功能查找
- 或者在复习过程中标记为不认识，系统会缩短该单词的复习间隔

### 4. 程序运行出现错误怎么办？
- 检查Python环境是否正确安装
- 确认所有依赖包是否已安装
- 查看错误信息并根据提示解决
- 如果问题无法解决，可以提交issue或联系开发者

### 核心模块说明

#### word_manager.py
负责单词的增删改查等基本操作，以及数据的持久化存储。

#### scheduler.py
实现艾宾浩斯记忆曲线算法，安排单词的复习时间。

#### utils.py
包含通用的工具函数，如菜单显示、用户输入处理等。

#### data_manager.py
提供高级数据管理功能，包括导入导出、备份恢复等。
